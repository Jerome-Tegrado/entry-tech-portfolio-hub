# Entry Tech Portfolio Hub

Proof-first portfolio projects for entry-level cloud and technical support roles.

## What This Repo Shows

This repository is organized as a Fundamentals track with four completed projects. Each project includes documentation, validation records, and evidence artifacts.

Target roles:
- IT Support (cloud-leaning)
- Technical Support Specialist / Engineer
- Cloud Support Associate / Junior Cloud Support
- Microsoft 365 / Entra ID Administrator
- Access / Identity Support

## Design Principles

- Evidence over claims: outputs, screenshots, and artifacts are required.
- No filler work: each project should be resume- and interview-ready.
- Support flow clarity: diagnose, validate, document, and summarize.
- Cost discipline: temporary lab resources are cleaned up after validation.

## How To Use This Repo

1. Open `start-here/fundamentals-index.md`.
2. Review each completed project under `fundamentals/`.
3. Reuse `templates/` to build additional projects in the same format.
